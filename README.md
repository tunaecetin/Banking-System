🏦 Banking System (Java)

A console-based Banking System written in Java. This project simulates
customers, accounts, transactions, and financial portfolios.

------------------------------------------------------------------------

FEATURES

-   Individual & Institutional Customer management
-   Savings & Checking accounts
-   Deposit, Withdraw, Transfer operations
-   Transaction history tracking
-   Interest calculation for accounts
-   Portfolio management (A & B portfolios)
-   Account comparison & portfolio comparison
-   Secure login with keycode

------------------------------------------------------------------------

-   Two account types: Savings Account Checking Account
